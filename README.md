# XNVezi2-MP
Open source for image editing full editor.
# Studio instructions
Full Studio editor — single HTML file, no placeholders. Here's what's included:

**File Menu** — Upload Image, Export as PNG / JPG / JPEG / TIFF / SVG / XCF / PSD

**Edit Menu** — Undo / Redo / Copy / Paste (+ Ctrl+Z/Y/C/V keyboard shortcuts)

**Effects Menu** — All shaders grouped by category (Color, Light, Distort, Blur, Sharpen, Texture, Generate, Color Space, 3D), all implemented as real WebGL GLSL fragment shaders

**Plugins Menu** — Create Filter/Effect as JSON+GLSL, Create Plugin as JSON, Save/Load filters

**UI Functions** — `UI()`, `UImenu()`, `slider()`, `cubefile()`, `point()`, `marker()`, `boolean()`, `color()`

**Layer/Edit Functions** — `unlimitedlayers()`, `rotate()`, `blendmodes()`, `scale()`, `positionxandy()`, `uploadimage()`, `fonts()`, `exportimage()`

**Right Panel** — Transform, Blend Modes, Opacity, Font picker, RGB Color sliders, Histogram, Applied Effects list, History stack, Unlimited Layers with visibility toggle
